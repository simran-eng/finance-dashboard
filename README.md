Finance Dashboard

This project is a simple and interactive finance dashboard built using React and Tailwind CSS. It provides a clear overview of financial data through visualizations and structured components.

Features

Summary cards displaying balance, income, and expenses
Pie chart for category-wise expense distribution
Line chart for time-based financial trends
Transactions table with date, amount, category, and type
Search functionality to filter transactions
Role-based interface:
Viewer: can view data
Admin: can delete transactions
Insights section highlighting the highest spending category

Tech Stack

React (Vite)
Tailwind CSS
Recharts

Approach

The goal of this project was to create a clean and intuitive user interface for managing and visualizing financial data. The focus was on simplicity, usability, and modular component structure using React.

Setup Instructions

Install dependencies:
npm install

Run the development server:
npm run dev

Open in browser:
http://localhost:5173

Notes

The project uses mock data
No backend is included
Designed as a frontend assignment project
